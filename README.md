## Installation
  To install necessary dependencies, run the following command:
  ```
  npm i
  ```
  ```
  npm start
  ```

## Usage
  The end user may need a .env file with a session key for salt hashing.

## Technologies
* React
* MongoDB / Mongoose
* AJAX
* Passport.js
* Bcrypt
* Material UI
